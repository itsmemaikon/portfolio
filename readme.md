# 💼 Portfólio Pessoal — Maikon Lopes

## 📌 Sobre o Projeto
Este repositório contém o código-fonte de um portfólio pessoal desenvolvido para apresentar projetos, habilidades e informações de contato.

O objetivo principal é servir como uma vitrine profissional, destacando experiências na área de Análise de Sistemas, desenvolvimento e integração de soluções.

---

## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
- Ion Icons (ícones)

---

## 📂 Estrutura do Projeto
```
portfolio-main/
│
├── index.html          # Estrutura principal da página
├── style.css           # Estilização do layout
├── script.js           # Interações e comportamento
│
├── assets/
│   ├── images/         # Imagens do site
│   └── projects/       # Imagens dos projetos
│
└── readme.md           # Documentação original
```

---

## 🖥️ Funcionalidades
- Navegação por seções (scroll suave)
- Seções organizadas:
  - Sobre
  - Habilidades
  - Projetos
  - Contato
- Exibição de projetos com imagens
- Links diretos para:
  - GitHub
  - LinkedIn
  - E-mail
- Layout responsivo

---

## 📸 Seções do Portfólio

### 🏠 Home

### 🧠 Habilidades

### 💻 Projetos
Demonstração de projetos desenvolvidos com foco prático.

### 📬 Contato
Formas de entrar em contato e redes profissionais.

## ▶️ Como Executar o Projeto
1. Clone o repositório:
```bash
git clone https://github.com/itsmemaikon/portfolio.git
```

2. Acesse a pasta:
```bash
cd portfolio-main
```

3. Abra o arquivo `index.html` no navegador
